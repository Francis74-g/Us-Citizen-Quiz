U.S. Citizenship Quiz App
A responsive JavaScript web application that helps users practice for the U.S. Citizenship (Civics) Test.
1. Features
 
- ✅ 100+ U.S. citizenship questions (2026-ready)
- 📱 Fully responsive (mobile, tablet, desktop)
- 🔘 Multiple-choice answers
- 🎯 Instant feedback:
  - Green = correct
  - Red = incorrect
- 📊 Score tracking
- 📈 Progress indicator
- 🔄 Restart quiz option
- 🌐 Can be installed as a Progressive Web App (PWA) 

2. Project Structure
/project-folder
│── index.html # Structure of the app
│── styles.css # Styling and responsive design
│── script.js # Quiz logic and questions
│── manifest. Json # PWA configuration (optional)
│── service-worker.js # Offline support (optional)
# 🇺🇸 U.S. Citizenship Quiz App

 3. How to Run the App
1. Download or clone the project
2. Open the folder
3. Double-click `index.html` OR open in a browser
No installation required.

4. Mobile Friendly
This app is fully responsive and works on:
 
- Smartphones
- Tablets
- Laptops
- Desktop computers
 

5. Install as an App (Optional)
 
hosted online by NETLIFY:
1. Open the app in Chrome/Edge
2. Click **"Add to Home Screen"** or **"Install App"**
3. Use it like a native mobile app
